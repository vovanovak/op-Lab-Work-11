Побудувати дерево, що відображає формулу ((a+b)*c-d), де коренем
дерева та його підкоренями є операції  +, -, *, /, а листками є змінні a, b, c, d .
Вивести значення дерева-формули. Надрукувати відповідні піддерева у1=а+b,
у2=у1*с, y3=y2-d.

1. ПОЧАТОК
2. Вводимо коефіціенти a, b, c, d
3. Ініціалізуємо дерево
	3.1 Додаємо корінь дерева знак "-"
	3.2 Ініціалізуємо правий вказівник кореня, яке відповідає числу d
	і лівий вказівник, якому відповідає вираз (a+b)*c
	3.3 Ініціалізуємо вузол, який відповідає за вираз (a + b) * c
		3.3.1 Лівий вказівник вузла буде представляти вираз (a + b), 
		правий вказівник буде листком з числом с, значення вузла буде = "*"
			3.3.1.1 Ініціалізуємо вузол який відповідає за вираз
			(a + b): значення вузла буде = "+", лівий вказівник буде посилатись
			на листок зі значенням а, а правий на листок зі значенням b
4. Обраховуємо значення виразу
	4.1 Викликаємо рекурсивну функцію count
		4.1.1 ЯКЩО даний вузол є листком, повертаємо значення листка
		4.1.2 ІНАКШЕ 
			4.1.2.1 Викликаємо функцію count для лівого вказівника на вузол
			4.1.2.2 Викликаємо функцію count для правого вказівника на вузол
			4.1.2.3 ЯКЩО значення вузла = "+":
				4.1.2.3.1 Результат := значення count для лівого вказівника +
				 значення count для правого вказівника 
			4.1.2.4 ЯКЩО значення вузла = "-":
				 4.1.2.4.1 Результат := значення count для лівого вказівника -
				 значення count для правого вказівника 
			4.1.2.5 ЯКЩО значення вузла = "*":
				 4.1.2.5.1 Результат := значення count для лівого вказівника *
				 значення count для правого вказівника 
			4.1.2.6 ЯКЩО значення вузла = "/":
				 4.1.2.6.1 Результат := значення count для лівого вказівника /
				 значення count для правого вказівника 
			4.1.2.7 Повертаємо результат
5. Виводимо результат
